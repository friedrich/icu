 1. Open a command window
 2. set ICU_ROOT=<path to ICU root directory>
 3. set XLIFF_JAR_DIR=<path to xliff.jar>.
    To build xliff.jar download ICU4J and do
    ant xliff.
 4. set JAVA_HOME=c:\jdk1.4.2_04
 5. devenv
 6. Open the solution and build it
